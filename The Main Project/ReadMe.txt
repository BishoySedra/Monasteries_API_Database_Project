Steps to open the project :
1- Open SQL Server Management Studio .
2- Restore database .
3- Open Visual Studio EnterPrise .
4- Open the solution of the project .
5- Set as Startup Project on WebView Project and run it without Debug Mode by click (Ctrl + f5) .
6- Set as Startup Project on WebAPI Project and run it with Debug Mode by click (f5) .
7- Open the browser to view the project .
